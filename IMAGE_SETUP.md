# Image Setup for 0xConfess

## Required Image
Please save the Web3 wallet illustration image to:
```
client/public/images/0xconfess-hero.png
```

The image you provided shows a purple/blue Web3 wallet with blockchain elements. This will be displayed on the homepage hero section.

### Steps:
1. Download or save the attached image as `0xconfess-hero.png`
2. Place it in the `client/public/images/` directory
3. Reload the page at http://localhost:8080

The HomePage component is already configured to use this image at `/images/0xconfess-hero.png`.
